    macOS更新后 终端提示：The default interactive shell is now zsh.

    作者:J.sky | 发表于:2023-06-02 10:17:13 | 更新于:2023-06-02 11:30:06 | Tag: macOS

    昨天升级了macOS系统到macOS Monterey,之后每次打开终端都会提示：

    The default interactive shell is now zsh.
To update your account to use zsh, please run `chsh -s /bin/zsh`.

    其实早就不怎么使用zsh了，正好卸载了吧。

    sudo rm -rf /usr/local/bin/zsh

    试了一下，但是还是有烦人的提示！

    解决办法

    后来搜的一个方法：

    chsh -s /bin/zsh

    这个还是不管用的！！

    最终解决方法：

    在命令行输入：

    vim ~/.bash_profile

    在文件内输入

    export BASH_SILENCE_DEPRECATION_WARNING=1

    重启终端，完美解决！
