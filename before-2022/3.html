    python3 Markdown模块解析Markdown代码

    作者:J.sky | 发表于:2017-08-19 19:59:46 | 更新于:2023-04-13 09:40:24 | Tag: Python基础

    python3 markdown模块解析markdown代码，blog中需要一个解析markdown代码的模块，测试了一下markdown模块 ，简单好用。

    安装模块

    pip3 install markdown

    使用

    markdown模块的使用非常简单了，参考下面的代码：

    
#codeing=utf-8
#引入markdown解析模块
from markdown import markdown

s = '###这里是标题'
m = markdown(s)#解析markdown文件
print(m)
